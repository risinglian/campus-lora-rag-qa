# 微调前后同题对比样例

来源：`eval_results/compare_12_latest.json`，为已完成的 12 题 Base / LoRA / LoRA+RAG 三模式真实输出中抽取。

## 问题：校园卡丢了怎么办？

### Base 输出

如果校园卡丢失了，可以采取以下步骤：

1. 立即联系校方服务热线或辅导员报告情况。
2. 在网上或通过其他方式补办新卡。
3. 保留所有相关证明材料的复印件。

请务必及时处理，以免影响学习和生活。

### LoRA 输出

可到服务窗口挂失或补办新卡，旧卡按流程注销。 如遇特殊情况，请以学校最新通知或相关部门答复为准。

### LoRA+RAG 输出

请第一时间在校园卡服务平台或服务大厅挂失，随后按流程补办新卡。 如遇特殊情况，请以学校最新通知为准。

检索命中：校园卡服务知识；网络服务知识；医疗服务知识

### 分析
- Base 的特点：能给出报告、补办等通用建议，但没有明确校园卡平台或服务大厅。
- LoRA 的变化：明确提到挂失、补办和旧卡注销，回答更接近校园服务口径。
- LoRA+RAG 的提升：检索到校园卡服务知识后，回答更直接地说明先挂失再补办。

## 问题：如果错过选课时间，还有没有补救办法？

### Base 输出

如果错过了选课时间，通常情况下是没有补救办法的。大多数大学和学院都有明确的时间表，一旦过了截止日期，就无法再进行选课操作了。建议在规定时间内完成所有必要的步骤，以免错过任何重要的课程安排。如果有特殊情况需要调整，请及时联系学校的教务处或相关部门寻求帮助。

### LoRA 输出

一般需在下学期开学后重新注册和选课。 如遇特殊情况，请以学校最新通知或相关部门答复为准。

### LoRA+RAG 输出

发现选课时间冲突时，应优先联系开课学院或教务处，按补退选流程处理。 如遇特殊情况，请以学校最新通知或相关部门答复为准。

检索命中：教务服务知识；图书馆服务知识；校园卡服务知识

### 分析
- Base 的特点：能给出联系教务等通用方向，但倾向于判断“没有补救办法”。
- LoRA 的变化：回答更像校园问答格式，但该题单独 LoRA 给出的处理路径不够准确。
- LoRA+RAG 的提升：检索教务服务知识后，能明确联系开课学院或教务处并按补退选流程处理。

## 问题：校园卡丢了并且宿舍灯坏了，我分别应该怎么处理？

### Base 输出

1. 校园卡丢了：首先报告给辅导员或相关部门，说明情况并申请补办；其次在校园内寻找丢失的卡片，如果找不到可以联系保卫处寻求帮助。
2. 宿舍灯坏了：联系宿管老师或后勤部门报修，告知他们灯具损坏的情况，他们会安排人员进行维修。同时，注意检查是否有其他设施也出现故障，及时向相关部门反映。

### LoRA 输出

先挂失校园卡，再报修宿舍灯。 如遇特殊情况，请以学校最新通知或相关部门答复为准。

### LoRA+RAG 输出

先挂失校园卡，再报修宿舍灯。 如遇特殊情况，请以学校最新通知或相关部门答复为准。

检索命中：宿舍服务知识；网络服务知识；校园卡服务知识

### 分析
- Base 的特点：能分别给出校园卡和宿舍灯的通用处理建议，但校园卡路径偏泛化。
- LoRA 的变化：回答压缩为“先挂失校园卡，再报修宿舍灯”，更像校园服务流程摘要。
- LoRA+RAG 的提升：检索到宿舍服务、网络服务和校园卡服务知识，能支撑两个事项的处理方向。
